# Сравнение инвестиционной привлекательности GSTD vs Amazon AWS / Google Cloud

**Executive Summary**: GSTD offers a decentralized alternative to traditional cloud providers, leveraging idle consumer hardware to provide compute power at **40-60% lower cost** while maintaining high availability through redundancy.

## Ключевая ценность: Цена и Эффективность

Традиционные децентрализованные сети (DePIN) предлагают экономию за счет использования простаивающего оборудования. GSTD Platform идет дальше, интегрируя **GSTD (Guaranteed Service Time Depth)** — метрику качества, гарантирующую выполнение задач с подтвержденной надежностью.

| Метрика | Amazon AWS (t3.medium) | Google Cloud (e2-standard-2) | GSTD Platform (Node Grade A) | Экономия для Заказчика |
| :--- | :--- | :--- | :--- | :--- |
| **Стоимость (в час)** | $0.0416 | $0.0670 | **$0.0200** | **~52-70%** |
| **Модель Оплаты** | Pay-as-you-go | Pay-as-you-go | **Pay-for-result** | Оплата только за валидный результат |
| **SLA / Гарантии** | 99.9% Uptime | 99.9% Uptime | **Cryptographic Proof** | Математическая гарантия выполнения |
| **Data Transfer** | Expensive ($0.09/GB) | **Free** (Included) | **Infinite ROI** |
| **Storage** | EBS costs extra | Ephemeral included | Savings |
| **Commitment** | None (On-Demand) | None | Equal |

**Investor Takeaway**: GSTD undercuts the market leader by >50% by eliminating data center overhead (cooling, rent, security).

## 2. Модель работы

### Amazon AWS
*   **Centralized**: Single point of failure (Region outages).
*   **KYC/Corporate**: Requires credit cards, address verification, account approval.
*   **Hardware**: Proprietary data centers.
*   **Green Score**: Medium (Improving, but massive energy consumption).

### GSTD Network
*   **Decentralized**: Resilient. If one node fails, task reroutes instantly.
*   **Permissionless**: Connect with a TON wallet. No names, no cards.
*   **Hardware**: Global grid of existing consumer devices (Circular Economy).
*   **Green Score**: **High** (Utilizes electricity already spent/idle).

## 3. Market Opportunity

*   **Target Audience**: AI Researchers, Student Developers, crypto-native dApps, Scrapers/Crawlers.
*   **Market Gap**: High demand for cheap, short-term compute (Spot Instances) which is often unavailable or complex on AWS.
*   **Unique Selling Point**: **Geographic Diversity**. A user can run code specifically in "Almaty" or "Residential IP" which is impossible with standard cloud providers.

## 4. Growth Projection (TFLOPS)

*   **Current**: ~15.2 TFLOPS (Bootstrap phase)
*   **Q3 2026**: 500 TFLOPS (Projected 1,000 active nodes)
*   **Q4 2026**: 2,500 TFLOPS (GPU Integration)

## 5. Conclusion

GSTD is not trying to replace AWS for banking infrastructure. It is disrupting the **flexible, ephemeral compute market** (Docker containers, CI/CD jobs, AI inference).

**Verdict**: Strong Buy / Early Stage Infrastructure Play.
